# How is this book
好書客，一個分享讀書心得的平台。

## 網站需求：

平台提供書籍列表與內容，供匿名/會員（待討論）讀者發表讀書心得。

不同於求職天眼通，「好書客」提供一個平台，讓讀者反映真實的書籍心得。

讀書心得可為：
1. 好評 & 壞評
2. 書籍難易度
3. 適合什麼階段的人去看
4. 適合什麼職業

EX: Android 程式設計實例入門 這本書
1. 適合初學者
2. 適合學生、手機開發者等職業


## 技術規格：

書籍資料來源：
1. 串接書籍網站 API （博客來、天龍等）。
2. 寫爬蟲去爬資料（假設沒辦法串接 API 的話）。

後端：Apache + Laravel + PHP7。

前端：未定。
